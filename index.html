Triangle Strip Generator

Description:

This is a javascript based generative web application. Please use the controls on the right to manipulate the variables.

Parameters:

  • Numpoints: This controls the number of points in each individual triangle strip element.
  • Stroke: This controls the weight of the stroke in each individual triangle strip element.
  • R: This controls the amount of red in each individual triangle strip element.
    G: This controls the amount of green in each individual triangle strip element.
    B: This controls the amount of blue in each individual triangle strip element.
    Alpha: This controls opacity of each individual triangle strip element.
    Color: This controls the color of the stroke in each individual triangle strip element.
    bColor: This controls the color of the window background.
